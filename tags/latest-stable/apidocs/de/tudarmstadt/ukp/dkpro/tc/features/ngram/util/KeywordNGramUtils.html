        Finds all minN- to maxN-length ngrams of tokens occurring in the keyword list. All tokens should already be lowercased, if applicable. The keyword list can contain multi-token words like "Brussel sprouts". If keyword list contains both "Brussel" and "Brussel sprouts", then only "Brussel sprouts" will be added. Otherwise, the smallest multiword matching keyword will be added.
