    public class LuceneNGramCPFE
extends LuceneNGramPFE
implements PairFeatureExtractor
    Combination pair ngram feature extractor. Creates features that are combinations of ngrams from both documents in the pair. Returns all combinations of qualified ngrams from each of the two documents. A feature value is 1 if each of the ngrams appeared in their respective text, and 0 otherwise.
    For example, given two documents:
    Document 1: "Cats eat mice"
    Document 2: "Birds chase cats"

    some combination ngrams are:
    comboNG_cats_birds:1, comboNG_cats_eat_birds:1, comboNG_cats_birds_chase:1, etc.
    Note: To extract ngrams from a pair of documents that are not combinations of ngrams across the documents, please use LuceneNGramPFE.
    Author:
